## Etapes d'utilisation :

Pour pouvoir profiter des gems, n'oublie pas un petit :
```
   $> bundle install
```

Ensuite :
```
   $> rails db:create
```

Enfin :
```
   $> rails db:migrate
```

Dernière étape pour ceux qui veulent le voir en local mais sinon le lien heroku marche !!! :boom:

Et pour finir :
```
   $> rails server
```
